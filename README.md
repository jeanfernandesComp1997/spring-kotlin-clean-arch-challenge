# spring-kotlin-clean-arch-challenge
Spring kotlin clean arch challenge with webflux and Coroutines

## Pre requisites
* Java 17+
* Wiremock

## Technologies used
* Java 17
* Spring Boot 3.2.2
* Kotlin
* H2 memory database
* SMTP
* Http
* Wiremock

## Local environment setup
Set this environments variables:

* SMTP_HOST 
* SMTP_PORT 
* SMTP_USER 
* SMTP_PASS
